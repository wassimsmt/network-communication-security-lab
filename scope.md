# Scope

## In Scope
- Kali Linux VM (tester)
- Ubuntu Server VM (target)
- Host-only network segment between VMs
- Services: SSH (22/tcp), HTTP (80/tcp)
- Tools: nmap, ss, tcpdump, ufw

## Out of Scope
- Attacks against real/production systems
- Brute-force, exploit development, malware
- Internet-facing scanning outside the lab

## Rules
- Controlled lab only
- No persistence/backdoors
- Evidence collected via logs, scans, and packet capture
